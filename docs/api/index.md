## API Reference

This section of the documentation outlines the entire C++ codebase.

Each section (Data, Levels, Util) corresponds to a designated folder in the codebase, except the Core section, which includes documentation of all files that are not in one of the specified subfolders.

### Data

The **Data** section contains the code defining the key data structures underlying telemetry management:

- ObjectInfo
- ObjectInfoStore
- Timeline

Read the documentation explaining these data structures [here](../design-docs/object-state-tracking/index.md).

### Level

The **Level** section covers systems related to level placement (outer space & planetary).

### Util

The **Util** section includes utilities for planet radius calculation & other tasks.

### Core

The **Core** section deals with everything else.
